# セキュリティポリシー

## セキュリティ問題の報告方法

このプロジェクト（個人運用・研究用途寄り）におけるセキュリティ上の脆弱性を発見した場合は、**絶対に公開Issueに詳細を記載しないでください**。

代わりに、以下の安全な方法でご連絡をお願いいたします。

1. **GitHub Security Advisory 機能** を利用する。
2. もしくは、GitHubアカウントに紐づく連絡先まで非公開の手段でお知らせください。

ご報告いただいた内容については、確認と修正ができ次第、迅速に対応いたします。

## 注意事項

- Issue等で質問やバグレポートを作成する際、誤って**ご自身のAPIキーやシークレット情報**を含めないよう、十分ご注意ください。
- 当プロジェクトは個人および研究目的で公開しており、商用のSLAや24時間体制でのサポートを保証するものではありません。あらかじめご了承ください。
